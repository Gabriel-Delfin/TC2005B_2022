1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Java es un lenguaje de programación orientada a objetos y Javascript es un lenguaje de scripts orientada a objetos, de la misma manera que Javascript solo puede ser ejecutado en HTMLs.


2. ¿Qué métodos tiene el objeto DATE?
  • FullYear: Año con 4 dígito
  • Month: Número del mes
  • Date: Día del mes
  • Day: Día de la semana
  • Hours: Hora actual
  • Minutes: Minutos actuales

    • 3. ¿Qué métodos tienen los arreglos?
  • Push: Inserta elemento al final del arreglo
  • Unshift: Inserta elemento al inicio del arreglo
  • Pop: Elimina un elemento del final del arreglo
  • Shift: Elimina un elemento del inicio del arreglo
  • Slice: Crea una copia del arreglo
